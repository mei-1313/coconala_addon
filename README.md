# ココナラ トーク履歴保存アドオン (Coconala Talkroom Saver)

ココナラのトークルーム（`https://coconala.com/talkrooms/*`）内のメッセージ履歴を解析し、送信者・送信ラベル・投稿日時・メッセージ本文（およびメッセージ内リンク）を抽出し、ワンクリックでJSON形式のファイルとして一括ダウンロードできるブラウザ拡張機能（Manifest V3対応）です。

画面の操作性を損なわない、最下部に固定されるGlassmorphism（グラスモルフィズム）スタイルの美しい固定フッターUIを備えています。

---

## 📂 ファイル構成

- **`manifest.json`**  
  ブラウザ拡張機能のメタデータ、動作対象URL、適用スクリプトなどを定義するManifest V3設定ファイル。
- **`content.js`**  
  ココナラトークルームのメッセージDOMをパースし、既読表示などのノイズを除去してJSONとしてパッケージングし、ダウンロードするメイン処理スクリプト。
- **`content.css`**  
  ココナラの画面に馴染みつつ高級感のある視覚的フィードバック（ボタンホバー、ローディング、完了表示アニメーション）を適用した固定フッターCSS。
- **`icons/`**  
  ChromeやFirefoxの拡張機能マネージャー等で表示されるココナラ調のグリーンをベースにした高解像度アイコン（16x16, 48x48, 128x128）。
- **`test_environment.html`**  
  ココナラの実機環境がなくても、ローカルでアドオンの動作とUIデザインを即座に確認できるモックテスト用HTMLページ。

---

## 🛠️ インストール手順 (Google Chrome / Edge 等)

1. ブラウザのアドレスバーに `chrome://extensions/` を入力して拡張機能管理ページを開きます。
2. 右上にある **「デベロッパー モード」** をオンにします。
3. 左上に表示される **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
4. このプロジェクトのルートフォルダ（`coconala_addon`）を選択します。
5. インストールが完了し、トークルームページでアドオンが自動で起動するようになります。

---

## 🧪 ローカルでの動作確認方法

アドオンを読み込まずとも、以下の手順で動作をローカル検証できます。

1. **`test_environment.html`** をブラウザ（Chrome等）で直接開きます。
2. 画面最下部に表示される **「Coconala Talkroom Saver」の固定フッター** を確認します。
3. **「JSON保存」** ボタンをクリックします。
4. `coconala_talkroom_unknown_YYYYMMDD_HHMMSS.json` がダウンロードされ、中身に解析されたメッセージデータが正しく保存されていることを確認します。
